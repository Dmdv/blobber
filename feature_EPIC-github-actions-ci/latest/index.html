There are [many](https://github.com/herumi/bls) / [libraries](https://github.com/herumi/bls-wasm) to help you generate this key. **BLS BN254 public key hex string**: The public key calculated from the previously generated private key, formatted as a hex string. **Client/wallet id**: A SHA3-256 hash of the public key **hex dump** - for an example of how this hex dump can be generated, see the pre-request script of this request in postman. A consensus of 20% is required from the miners, after which you can be sure that your request will be propagated.
